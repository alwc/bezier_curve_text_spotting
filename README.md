# Run Demo

```
bash vis_demo.sh
```

We assume that your symlinked `datasets/totaltext` directory has the following structure:

```
totaltext
|_ test_images
|  |_ 0000000.jpg
|  |_ ...
|  |_0000299.jpg
|_ annotations
|  |_ total_test.json 
```
